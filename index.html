opnsense-plugins - OPNsense FreeBSD pkg Repository

Built automatically from GitHub on every push to master.

Setup

fetch -o /usr/local/etc/pkg/repos/chefkoch-de42-plugins.conf \
  https://chefkoch-de42.github.io/opnsense-plugins/packages/opnsense-plugins.conf

pkg update && pkg install os-wancarp
