opnsense-plugins – OPNsense FreeBSD pkg Repository

Built automatically from GitHub on every push to master.

Setup

# 1. Add repository (covers all plugins below)
fetch -o /usr/local/etc/pkg/repos/chefkoch-de42-plugins.conf   https://chefkoch-de42.github.io/opnsense-plugins/packages/opnsense-plugins.conf

# 2. Install a plugin, e.g.:
pkg update && pkg install os-wancarp

Repository URL

https://chefkoch-de42.github.io/opnsense-plugins/packages

Available plugins

PackageVersionInstallRepo config
